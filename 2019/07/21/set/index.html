set

set 应该是基于平衡树的一种数据结构反正我不会写。值得注意的是set是会排序的，可以通过重载运算符的方法来更改排序方式

还有set中不允许有相同元素

它支持

  • set.begin() 返回指向第一个元素的迭代器
  • set.end() 返回指向最后一个元素的下一个元素 的迭代器
  • set.empty() 判断集合是否为空
  • set.size() 集合中元素的个数
  • set.insert() 在集合中插入元素
  • set.erase(x) 删除集合中等于$x$的元素
  • set.clear() 清除所有元素
  • set.find(x) 找到值为x的一个元素的迭代器
  • set.lower_bound() 返回指向大于（或等于）某值的第一个元素的迭代器
  • set.upper_bound() 返回指向小于（或等于）某值的第一个元素的迭代器
  • swap() 交换两个集合
